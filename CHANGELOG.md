# Changelog

All notable changes to EpsilonLab are documented in this file.

## v1.0.0 — 2026-03-04

Initial public release.

### Features

- **Laplace & Gaussian mechanisms** with configurable ε, δ, sensitivity, and seed
- **Rust/WASM computation engine** — all DP math runs client-side
- **13 curated presets** across Laplace, Gaussian, and Composition categories
- **Interactive charts** — noise PDF, noisy-results histogram, utility-vs-epsilon curve
- **Classroom Pack PDF** — multi-page handout generator for instructors
- **LMS embed mode** — iframe-friendly route with minimal chrome
- **URL-state sharing** — every simulator configuration is a copyable link
- **Student & Academic modes** — toggle detail level for different audiences
- **Privacy Accounting** — sequential composition budget tracker
- **CSV dataset upload** — bring your own data
- **SEO & outreach** — press page, sitemap, robots.txt
- **Diagnostics & health** — /health page, Shift+D diagnostics panel, error boundary
- **EmailJS contact form** — no backend required
